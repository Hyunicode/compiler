# compiler

记录一下《编译原理》的几个实验

1. RE to NFA

   使用Thompson构造算法实现正则表达式转换为NFA

2. NFA to DFA

   使用subset构造算法和state-minimization算法实现NFA转换为DFA

3. scanner

   ~~~
   // decaf语言的词法分析
   INPUT：decaf Source Code
   OUTPUT: Sequence of tokens
   ~~~

4. parser

   ~~~
   // decaf语言的语法分析
   INPUT：Sequence of tokens
   OUTPUT: Parse tree
   ~~~
